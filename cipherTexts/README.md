This folder contains a collection of ciphertexts, initially sourced from [The Code Book by Simon Singh](https://www.amazon.co.uk/Code-Book-Secret-History-Code-breaking/dp/1857028899). These texts will be explored and used as reference material as the project evolves, helping demonstrate the design decisions and extensions of the repository.

All texts are sourced from [The CipherTexts simonsingh.net](https://simonsingh.net/cryptography/cipher-challenge/the-ciphertexts/)
